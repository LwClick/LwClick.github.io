  • String
    • 底层priavte final char value[]
    • 常量，线程安全
    • 对String类型进行改变，生成新的String，指针指向新String
    • 适合少量数据
  • ==与equals
    • ==
      • 判断两个对象的地址（基本数据类型比较值，引用数据类型比较内存地址）
    • equals
      • 判断两个对象是否相等
      • String中的equals方法是被重写的，Object的equals方法是比较对象的内存地址
  • final
    • 对基本数据类型，一旦初始化后不能修改
    • 对引用数据类型，一旦初始化不能再让其指向另一个对象
    • 修饰类，该类不能继承，所有方法被隐式的指为final方法
    • 对方法，锁定该方法，防止任何继承类修改它的含义，提高效率
    • 类中所有的private方法都被隐式的指定为final
